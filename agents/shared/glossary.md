# Glossary

- `SOOP`: the broadcast source platform being collected from
- `VOD`: video-on-demand content captured after live broadcast
- `summary payload`: normalized data structure produced from source content for later UI or publishing use
- `post job`: a persisted unit of content intended for review and possible publication
- `approval gate`: a required review step before dispatch
- `adapter`: the isolated module that knows how to talk to one publish target
- `artifact`: saved evidence such as sample JSON, logs, screenshots, or test output
- `handoff`: the written summary one role leaves for the next role
