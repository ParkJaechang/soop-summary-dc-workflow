# Project Template

This is the starter layout for future projects.

## Intent

- keep each project's workflow isolated
- reuse the same harness ideas
- avoid changing the current project's `C:\python\agents`

## First Edits

1. update `agents/shared/project_brief.md`
2. update `agents/shared/architecture.md`
3. update `agents/board/ownership.md`
4. set your first active task in `agents/board/tasks.yaml`
5. set a realistic estimate in `agents/board/progress.md`
